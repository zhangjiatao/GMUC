# Gaussian Metric Learning for Few-Shot Uncertain Knowledge Graph Completion

## Install
Make sure your local environment has the following installed:
```
Python3
Pytorch >= 1.5.0
visdom 
```

## Usage
start visdom
```
python -m visdom.server
```
run experiment
```
python run_exp.py
```
check result in 
```
http://localhost:8097/
```
